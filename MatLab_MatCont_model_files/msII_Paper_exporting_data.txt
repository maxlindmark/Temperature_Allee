%-- Note: Analyses are based on the default parameters above (and Table S2 in the Appendix) unless stated otherwise in the curve name. 
%-- Time integrations are not exported but are only saved, like this: o_"name", where the name is based on the specific parameter deviation from the default model and "o" refers to "orbit". 
% Simulated data is exported locally so you unfortunately need to specify your own path.
% The numbers in parenthesis after the curve name corresponds to (1) backward continuation (2) forward continuation (3) continuation from branch point to get stable C-R curve after predator extinction.
% The code below reads data from matcont, appends and reshapes data before saving.
% This is MATLAB code and should be copied into an editor or directly into the command window.

% Limit points: These points represent special points where a curve reaches a maximum or minimum parameter value and bends back on its self. When the curve represents equilibrium points of a problem in biological population dynamics such a point marks the threshold where a stable equilibrium state and an unstable state (saddle point) merge and disappear.
% Branch point: These points represent special points where 2 different solution curves intersect for a particular parameter value. Although in generic systems of equations this happens only rarely, in systems of equations stemming from biological population dynamics this occurs frequently. These points then represent so-called invasion points, the intersection between a curve representing a zero-valued equilibrium density of a population and a curve representing equilibria with a positive equilibrium density. 
% These comments come from: read me in Floor & de Roos 2017 Supplement
